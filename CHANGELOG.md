# Changelog

All notable changes to this project will be documented in this file.

## 0.0.1

- Initial release of the JUCE Color Picker extension.
- Added VS Code color decorators and color picker rewrites for literal JUCE `Colour` expressions in C++.
- Supported `fromRGB`, `fromRGBA`, `fromFloatRGBA`, `fromHSV`, `fromHSL`, direct constructors, float-alpha constructors, and packed 32-bit ARGB literals.
- Added packaging, local VSIX installation scripts, MIT license metadata, repository metadata, and test coverage.
